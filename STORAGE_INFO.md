# 로컬스토리지 및 쿠키 저장 기능 정리

## 1. 사용자 로그인 정보 (`userData`)

### 저장 위치

- **localStorage**: "로그인 상태 유지" 체크박스가 체크된 경우
- **sessionStorage**: "로그인 상태 유지" 체크박스가 체크되지 않은 경우

### 저장 형식

```json
{
  "email": "user@geekflex.com",
  "name": "사용자 이름",
  "loginTime": "2024-01-01T00:00:00.000Z",
  "rememberMe": true
}
```

### 저장 위치

- `js/login.js`
- `js/header.js`

### 사용 기능

- **로그인 상태 유지**: `rememberMe` 값에 따라 localStorage 또는 sessionStorage 선택
- **자동 로그아웃**: `rememberMe`가 `false`인 경우 24시간 후 자동 삭제
- **로그인 상태 확인**: 페이지 로드 시 자동 확인

### 삭제 시점

- 사용자가 로그아웃 버튼 클릭 시
- `rememberMe`가 `false`이고 로그인 후 24시간 경과 시
- 브라우저 데이터 삭제 시 (localStorage의 경우)

### 관련 파일

- `js/login.js`: 로그인 처리 및 저장
- `js/header.js`: 사용자 정보 관리 및 UI 업데이트

## 2. 검색 히스토리 (`searchHistory`)

### 저장 위치

- **localStorage**

### 저장 형식

```json
["검색어1", "검색어2", "검색어3", ...]
```

### 저장 데이터

- 검색어 문자열 배열
- 최대 10개까지만 저장 (오래된 항목 자동 삭제)
- 중복 검색어는 최신 위치로 이동

### 저장 위치

- `js/search.js`
- `js/config.js`

### 사용 기능

- **검색 히스토리 표시**: 검색 모달 열 때 최근 검색어 표시
- **빠른 재검색**: 히스토리 항목 클릭 시 즉시 검색
- **히스토리 삭제**: 개별 항목 삭제 가능

### 삭제 시점

- 사용자가 히스토리 항목의 삭제 버튼 클릭 시
- 최대 개수 초과 시 오래된 항목 자동 삭제
- 브라우저 데이터 삭제 시

### 관련 파일

- `js/search.js`: 검색 히스토리 관리
- `js/config.js`: 초기 상태 로드

## 3. 즐겨찾기 영화 목록 (`favorites`)

### 저장 위치

- **localStorage**

### 저장 형식

```json
[123, 456, 789, ...]
```

### 저장 데이터

- 영화 ID 배열 (숫자 배열)
- 영화 상세 페이지에서 "찜하기" 버튼 클릭 시 저장/삭제

### 저장 위치

- `js/movie-detail.js`

### 사용 기능

- **찜하기 토글**: 영화 상세 페이지에서 찜하기 버튼으로 추가/제거
- **상태 표시**: 이미 찜한 영화는 하트 아이콘이 채워진 상태로 표시
- **페이지 로드 시 상태 복원**: 영화 상세 페이지 로드 시 찜하기 상태 자동 확인

### 삭제 시점

- 사용자가 찜하기 버튼을 다시 클릭하여 제거 시
- 브라우저 데이터 삭제 시

### 관련 파일

- `js/movie-detail.js`: 즐겨찾기 관리

## 4. 시청 목록 (`watchlist`)

### 저장 위치

- **localStorage**

### 저장 형식

```json
[123, 456, 789, ...]
```

### 저장 데이터

- 영화 ID 배열 (숫자 배열)
- 영화 상세 페이지에서 "보고싶어요" 버튼 클릭 시 저장/삭제

### 저장 위치

- `js/movie-detail.js` (라인 277, 292, 324)

### 사용 기능

- **시청 목록 토글**: 영화 상세 페이지에서 보고싶어요 버튼으로 추가/제거
- **상태 표시**: 이미 시청 목록에 있는 영화는 버튼이 활성화된 상태로 표시
- **페이지 로드 시 상태 복원**: 영화 상세 페이지 로드 시 시청 목록 상태 자동 확인

### 삭제 시점

- 사용자가 보고싶어요 버튼을 다시 클릭하여 제거 시
- 브라우저 데이터 삭제 시

### 관련 파일

- `js/movie-detail.js`: 시청 목록 관리
