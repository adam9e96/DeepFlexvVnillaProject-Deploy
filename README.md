# GeekFlex - 영화 리뷰 플랫폼

> 효율적인 리뷰 작성과 한국 특화 서비스를 제공하는 영화 리뷰 플랫폼

## 프로젝트 소개

GeekFlex는 **효율적인 리뷰 작성**과 **한국 특화 서비스**를 특징으로 하는 영화 리뷰 플랫폼.

마우스 없이도 키보드만으로 모든 기능을 사용할 수 있으며, 한국 사용자를 위한 OTT 플랫폼 정보와 검열 정보를 제공.

### 핵심 특징

- **⌨️ 키보드 중심 UI**: Ctrl+K 검색, 단축키 지원
- **🇰🇷 한국 특화**: OTT 플랫폼 정보, 검열 정보 제공
- **📱 반응형 디자인**: 모바일, 태블릿, PC 모든 환경 지원
- **🎨 모던 UI/UX**: 다크 테마, 그라데이션, 애니메이션
- **🔄 실시간 데이터**: TMDB API 연동으로 최신 영화 정보

## 주요 기능

### 1. 영화 검색 및 필터링

- **빠른 검색**: Ctrl+K 단축키로 즉시 검색
- **다중 필터**: 장르, 평점, 정렬 옵션
- **실시간 결과**: 타이핑과 동시에 결과 표시

### 2. 다양한 뷰 모드

- **카드 뷰**: 시각적으로 아름다운 그리드 레이아웃
- **리스트 뷰**: 정보가 풍부한 리스트 형태

### 3. 카테고리별 영화 탐색

- **상영중인**: 현재 상영중인 최신 영화
- **인기있는 영화**: 인기 순위 영화
- **애니메이션**: 애니메이션 영화 전용
- **드라마**: 드라마 장르 영화

### 4. 한국 특화 서비스

- **OTT 플랫폼 정보**: Netflix, 왓챠, 디즈니+ 등
- **검열 정보**: 관람등급 자동 표시
- **한국어 지원**: 모든 텍스트 한국어화

## 🛠 기술 스택

### Frontend

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **JavaScript (ES6+)**: 모듈화, 비동기 처리
- **Font Awesome**: 아이콘 라이브러리

### API & 데이터

- **TMDB API**: 영화 데이터 제공
- **Fetch API**: 비동기 데이터 로딩
- **JSON**: 데이터 교환 형식

## 사용법

### 메인 페이지 (index.html)

#### 헤더 네비게이션

**카테고리 메뉴**:

- **상영중인** : 현재 상영 중인 최신 영화 목록 (TMDB API `now_playing` 엔드포인트)
- **인기있는 영화** : 인기 순위 영화 목록 (TMDB API `popular` 엔드포인트)
- **애니** : 애니메이션 장르 영화 필터링 (장르 ID 16)
- **드라마** : 드라마 장르 영화 필터링 (장르 ID 18)

**뷰 전환 버튼**:

- **그리드 뷰** (네모 아이콘): 카드 형태로 영화 포스터를 크게 표시 (기본 뷰)
- **리스트 뷰** (목록 아이콘): 가로형 리스트로 더 많은 정보 표시

**검색 기능**:

- 검색 버튼 클릭 또는 `Ctrl + K`로 검색 모달 열기
- 영화 제목, 감독, 배우로 검색
- 실시간 검색 결과 표시
- 검색 히스토리 저장 (최대 10개, 로컬 스토리지)
- 키보드 네비게이션 지원 (↑↓ 화살표, Enter 선택, Esc 닫기)

#### 사이드바

- 사이드바 토글 버튼으로 열기/닫기
- 헤더와 동일한 카테고리 메뉴
- 필터 기능: 장르, 평점, 정렬 옵션
- 단축키 안내 표시
- 로그인 버튼

#### 메인 컨텐츠

**필터 섹션**: 장르, 평점, 정렬 옵션

**영화 목록**:

- **그리드 뷰**: 포스터, 제목, 줄거리, 평점, 개봉일, 장르 태그, 관람등급, OTT 플랫폼 정보
- **리스트 뷰**: 좌측 포스터, 우측 상세 정보

**페이지네이션**: 이전/다음 페이지, 페이지 번호 버튼 (페이지당 6개 영화)

### 로그인 페이지 (login.html)

#### 로그인 폼

**이메일 입력**:

- 필수 항목, 이메일 형식 검증
- 실시간 유효성 검사

**비밀번호 입력**:

- 필수 항목, 최소 6자 이상
- 표시/숨기기 토글 버튼
- 실시간 유효성 검사

**로그인 상태 유지**:

- 체크 시: localStorage에 저장 (영구 로그인)
- 미체크 시: sessionStorage에 저장 (브라우저 종료 시 로그아웃)

**비밀번호 찾기**: 링크 클릭 (현재 시뮬레이션 단계)

#### 소셜 로그인

- **Google 로그인**: 버튼 클릭 (현재 "준비 중" 알림)
- **Kakao 로그인**: 버튼 클릭 (현재 "준비 중" 알림)

#### 로그인 처리

**사용법**:

1. 이메일과 비밀번호 입력
2. (선택) 로그인 상태 유지 체크
3. 로그인 버튼 클릭 또는 Enter 키 입력
4. 로딩 중 표시 (약 2초)
5. 성공 시 메인 페이지로 자동 이동

**테스트 계정** (현재 시뮬레이션):

- `admin@geekflex.com` / `password123`

**에러 처리**: 이메일 형식 오류, 비밀번호 길이 부족, 로그인 실패, 네트워크 오류

#### 자동 로그인 확인

- 이미 로그인된 상태면 메인 페이지로 자동 리다이렉트
- `rememberMe`가 false인 경우 24시간 후 자동 로그아웃

### 📝 회원가입 페이지 (signup.html)

#### 기본 정보 섹션

**아이디**: 필수, 영문/숫자 4-20자, 실시간 검증

**닉네임**: 필수, 2-20자, 실시간 검증

**이메일**: 필수, 이메일 형식, 실시간 검증

**비밀번호**: 필수, 8자 이상, 영문/숫자/특수문자 포함, 표시/숨기기 토글

**비밀번호 확인**: 필수, 비밀번호와 일치 확인

#### 프로필 정보 섹션

**프로필 이미지**: 선택, 이미지 파일만, 최대 5MB, 미리보기 제공

**자기소개**: 선택, 최대 300자, 실시간 글자 수 카운터

#### 권한 및 약관 섹션

**이용약관 및 개인정보처리방침 동의**: 필수, 링크 클릭하여 약관 확인

**마케팅 정보 수신 동의**: 선택

#### 회원가입 처리

**사용법**:

1. 모든 필수 항목 입력
2. 약관 동의 체크
3. 회원가입 버튼 클릭 또는 Enter 키 입력
4. 로딩 중 표시 (약 3초)
5. 성공 시 로그인 페이지로 자동 이동 (2초 후)

**테스트 중복 계정** (현재 시뮬레이션):

- `admin` (아이디)
- `admin@geekflex.com`, `user@geekflex.com`, `test@geekflex.com` (이메일)

### ⌨️ 키보드 단축키

#### 전역 단축키 (모든 페이지)

| 단축키     | 기능               | 적용 페이지 |
| ---------- | ------------------ | ----------- |
| `Ctrl + K` | 검색 모달 열기     | 메인 페이지 |
| `Escape`   | 모달/사이드바 닫기 | 메인 페이지 |

#### 검색 모달 단축키 (메인 페이지)

| 단축키   | 기능                       |
| -------- | -------------------------- |
| `↑`      | 검색 결과 위로 이동        |
| `↓`      | 검색 결과 아래로 이동      |
| `Enter`  | 선택된 검색 결과 항목 선택 |
| `Escape` | 검색 모달 닫기             |

#### 폼 단축키 (로그인/회원가입 페이지)

| 단축키   | 기능                                 | 적용 페이지      |
| -------- | ------------------------------------ | ---------------- |
| `Enter`  | 폼 제출 (입력 필드에 포커스 있을 때) | 로그인, 회원가입 |
| `Escape` | 에러 메시지 숨기기                   | 로그인, 회원가입 |
